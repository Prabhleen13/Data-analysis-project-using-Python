# 📊 Student Social Media Addiction Analysis

## 📌 Project Overview
This project analyzes student social media addiction patterns using data analysis techniques in Python.

## 🎯 Objectives
- Understand social media usage patterns
- Identify addiction trends among students
- Visualize behavioral insights

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📂 Dataset
The dataset contains information about students' social media usage and behavioral patterns.

## 📊 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization

## 🚀 How to Run
1. Open the Jupyter Notebook
2. Run all cells step by step

## 📌 Conclusion
The project highlights patterns of social media addiction and its impact on students.
